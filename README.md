# tinyMCE
tinyMCE wrapper for Spiral and sf.js

## Usage Example

## Installation

`composer require spiral/tinymce`
`spiral register spiral/tinymce`

## Update
`composer update spiral/tinymce`
`spiral publish spiral/tinymce`

## Local Development

### Installation

    npm install -g gulp
    npm install

### Building

    gulp build
